Adobe Illustrator Project

This Illustrator project is a collection of the stuff I really enjoyed working on. It includes my personal website design, a set of fun chat stickers (featuring frogs and cats), and a name card I designed for myself. I wanted the whole thing to reflect my personality and design style, so I kept it playful and creative. From sketching the characters to putting everything together in a cohesive look, this project was a great way for me to explore illustration and layout design in a way that feels uniquely mine.
